Implement a TypeScript TraceLogger in packages/shared that coord agents call to write JSONL trace events to .devsteps/traces/spider_traces.jsonl. The VS Code extension monitors this file via FSWatcher and pushes updates to the WebviewPanel via debounced postMessage. Preview card shows the latest (1) trace event. Detail tab shows the last 10. Affected: packages/shared/src/schemas/trace.ts (new), packages/shared/src/utils/trace-logger.ts (new), packages/extension/src/extension.ts, .devsteps/traces/
