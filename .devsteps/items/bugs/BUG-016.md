# TreeView Badges Missing - WRONG APPROACH!

## Research Findings

**CRITICAL DISCOVERY:** FileDecorationProvider funktioniert NUR für File Explorer, NICHT für custom TreeViews!

### Evidence:
1. Stack Overflow (https://stackoverflow.com/questions/74999287): "you can't" - rechts-ausgerichteter Text in TreeView nicht mit FileDecorationProvider möglich
2. GitHub Issue #166614: "Not able to use FileDecorationProvider for tree view item"  
3. VS Code API Docs: FileDecorationProvider ist für FILE decorations, nicht für TreeView items

### Real Solution:
Badges müssen über `TreeItem.description` Property gesetzt werden, nicht über FileDecorationProvider!

```typescript
treeItem.description = this.getStatusBadge(this.item.status);
```
