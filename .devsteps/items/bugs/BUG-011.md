# Bug: CLI Init Missing Workflow Prompts and Instructions

## Problem

When users run `devsteps init`, the CLI only deploys `.github/agents/devsteps.agent.md` but not the essential workflow files that guide users through the devsteps methodology.

## Missing Files

**Prompts (workflow guides):**
- `devsteps-plan-work.prompt.md`
- `devsteps-start-work.prompt.md`
- `devsteps-workflow.prompt.md`

**Instructions (standards):**
- `devsteps.instructions.md`
- `devsteps-code-standards.instructions.md`

## Current Behavior

**CLI (`packages/cli/src/commands/init.ts` line 450):**
```typescript
writeFileSync(join(githubAgentsDir, 'devsteps.agent.md'), devstepsAgent);
```

Only copies agent file.

## Expected Behavior

**MCP Server (`packages/mcp-server/src/handlers/devsteps-init.ts` lines 220-234):**
```typescript
// Copy devsteps instructions
const devstepsInstructionsSource = join(sourceGithubDir, 'instructions', 'devsteps.instructions.md');
if (existsSync(devstepsInstructionsSource)) {
  const devstepsInstructionsContent = readFileSync(devstepsInstructionsSource, 'utf8');
  writeFileSync(join(githubInstructionsDir, 'devsteps.instructions.md'), devstepsInstructionsContent);
}

// Copy devsteps prompts
const promptFiles = ['devsteps-plan-work.prompt.md', 'devsteps-start-work.prompt.md', 'devsteps-workflow.prompt.md'];
for (const promptFile of promptFiles) {
  const promptSource = join(sourceGithubDir, 'prompts', promptFile);
  if (existsSync(promptSource)) {
    const promptContent = readFileSync(promptSource, 'utf8');
    writeFileSync(join(githubPromptsDir, promptFile), promptContent);
  }
}
```

**Note:** MCP server currently copies `devsteps.instructions.md` but NOT `devsteps-code-standards.instructions.md` - CLI should copy BOTH.

## Impact

Users relying on CLI for project initialization don't get:
- **Workflow guides** (how to plan/start/execute work systematically)
- **Methodology standards** (DevSteps workflow rules)
- **Code quality standards** (file size limits, quality principles)

## Solution

Add file copying logic to CLI init command:
1. Create `.github/prompts/` and `.github/instructions/` directories
2. Copy 3 prompt files to prompts directory
3. Copy 2 instruction files to instructions directory
4. Update success message to list all copied files
