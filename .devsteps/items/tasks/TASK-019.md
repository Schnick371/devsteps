Created CONTRIBUTING.md with comprehensive contribution guidelines and DCO requirements.

- How to contribute (bugs, features, code)
- Development setup instructions (Node 20+, dependencies, build)

- Developer Certificate of Origin (DCO) with signing instructions

- Full DCO 1.1 text included
- Commit signing instructions (manual + automatic config)
- Legal compliance for contribution tracking

**Monorepo Guidance**:
- Package structure explained
